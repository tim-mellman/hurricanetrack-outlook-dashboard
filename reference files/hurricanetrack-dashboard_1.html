About This Dashboard
What this shows

This dashboard tracks Atlantic hurricane season outlooks from NOAA, the UK Met Office, Colorado State University (CSU), and ECMWF, comparing their predictions against verified post-season actuals going back to 2011.

Three categories are tracked: named storms (tropical storms + hurricanes), hurricanes, and major hurricanes (Category 3+). ACE (Accumulated Cyclone Energy) is also tracked as a single metric summarizing overall season activity.

Skill Scores

Mean Absolute Error (MAE) measures average forecast miss in each category. Bias shows systematic over- or under-forecasting. The skill table ranks sources across all verified years.

Activity Bands

ACE activity bands follow NOAA thresholds: below-normal (<75), near-normal (75–150), and above-normal (>150). These categories contextualise whether seasonal outlooks anticipated active or quiet years.

Data & Credits

Concept and graphics by HurricaneTrack. Historical actuals from NOAA's NHC. Climatological averages (1991–2020) from NOAA. This dashboard is updated manually; use the Admin panel to enter new outlooks as they are released each spring.
